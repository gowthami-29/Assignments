1. Role of Frontend:
The Frontend-is the part of a web application that users directly interact with.It is responsible for creating a user-frirndly and interactive experience.
## User Interface:
    Frontend designs and display the layout of the application, inclding buttons, forms, text, images,and navigation using html, css and java script.
## User Interaction:
    It handles user action such as clicking, typing, scrolling, and form submissions, and provides instant feedback like alerts and validations.
## Communication with Backend:
    Frontend sends requests to the backend and displays the data received from the server to the user.1
