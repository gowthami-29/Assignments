1. Role of Frontend:
The Frontend-is the part of a web application that users directly interact with.It is responsible for creating a user-frirndly and interactive experience.
## User Interface:
    Frontend designs and display the layout of the application, inclding buttons, forms, text, images,and navigation using html, css and java script.
## User Interaction:
    It handles user action such as clicking, typing, scrolling, and form submissions, and provides instant feedback like alerts and validations.
## Communication with Backend:
    Frontend sends requests to the backend and displays the data received from the server to the user.1

2. Role of Backend:
    It is a server-side part of web application that manages data and business operations.
## Server-side Processing:
    Backend process users requests, applies logic, and generates responses.
## Database Handling:
    It stores, retrueves, updates, and delete data from databases such as user details, orders, or products.
## Security and Authentication:
    Backend handles user authentication, authorization, data validation, and ensures secure acess to the application.
3. Business Logic:
    Business logic refers to the set of rules and decision making processes that define how an application works according to business requirements.
    ## Real-world examples:
    - E-commerce Website:
        Applying discounts, calculating total price, checking stock availability, and adding tax during checkouts.
4. Client-Server Model:
    - Client: the client the user's device or application that sends request to the server.
    - Server: the server is a computer that process client requests and sends back response.
    - Communication: Communication happens through the internet using protocolslike http/https, where the client sends a request and the server responds with data.
5. Three-Tier Architecture:
    Three-Tier Architecture is a web application design where the application is divided into three separate layers for better structure and maintainability.
    - Presentation layer:
        This is frontend layer.it handles ui and user interface.
    - Application layer:
        this layer contains business logic.it process user requests and applies rules.
    - Data layer:
        this data layer manages data storage and retrieval.it interacts with databases
